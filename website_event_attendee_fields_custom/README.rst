=========================================================
 Attendee registration: Birthdate, Passport, Nationality
=========================================================

Ask information on registration and stores at Partner record:

* First name
* Last name
* Email
* Nationality
* Passport
* Birthdate

Key features are provided by dependencies. See `<__manifest__.py>`_
