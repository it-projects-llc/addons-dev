Tickets purchasing: force user to sign in / sign up

User registration at your portal is always a plus for marketing

How It Works

You can install this module in a usual way. After installation you need to configure.

Open page Events on Website as non-authenticated user and select some Event.

You will see that to register for the event you now need to register or log in using the Sign In/Sign Up button (or Button Register Now if website_event_sale is installed)

Clicking on the button, you will be redirected to the login page, and then, after registration / authorization, redirected to the events page where you can now order tickets.
