Tickets purchasing: force user to sign in / sign up

User registration at your portal is always a plus for marketing

The module updates /event/EVENT-ID/register page when user is not authenticated.

How It Works

You can install this module in a usual way. After installation you need to configure.

Open page /event on Website as non-authenticated user and select some Event.

Button Register Now (Order Now if website_event_sale is installed) is renamed to ask for sign in / sign up.

As a result, if you click the button it opens login page sign in or sign up and then you are redirect back to event page and now you can order the tickets.
