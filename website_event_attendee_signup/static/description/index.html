Sign-up event attendees to portal

Attendees can use portal dashboard to make extra purchases for the event, for example.

The modules creates User from every Event registration send an email with signup url to access the portal.

The modules adds email template Event: Signup, which can be used directly or as an example to modify other email template.

How It Works

You can install this module in a usual way. After installation, you need to configure the module according to the documentation.

Register few attendees to the Event via Website (/event page).

Confirm the registrations (e.g. via backend).

As a result, each participant will receive an email with a link to register on the portal.
