Real Multi Website (blog extension)

Blocks an access to a website blog from websites which have not been specified as allowed for this blog

Key Features:
  • All features from Real Multi Website module
  • Blocks an access to a website blog from not allowed websites

How It Works

Let's say we have 2 websites:

  • example1.com
  • example2.com
Go to Website → Blog → Blogs menu and select at Allowed Websites the websites, where your blog should be available on, e.g. example1.com.

Open Website → Configuration → Menus and be sure that you have blog menu for one of your website at least, otherwise create a new one

Then open the blog on website like http://example1.com/blog/1. It's opened correctly. But if you try to open the same blog url from another website http://example2.com/blog/1, you will see access error "403: Forbidden"
