==================
 Deselect Partner
==================

Configuration
=============

* Go to ``Point of Sale >> Configuration >> Point of Sale``

  * Select a POS
  * Click on ``[Edit]``
  * Specify a **Customer Deselection Interval** in seconds
  * Click ``[Save]``

Usage
=====

* Open a POS
* Click ``Customers``
* Select a customer
* Click ``Set Customer``
* Wait time you set in configuration
* RESULT: Customer is Deselected
